# APECtyper GUI

Simple graphical user interface (GUI) for running **APECtyper.sh** without using the terminal.

This tool allows users to:
- Select a FASTA file
- Select an output directory
- Run APECtyper with a single click

Designed for Linux/macOS systems.

---

## Requirements

- Python ≥ 3.8
- Tkinter (usually included with Python)
- APECtyper.sh script
- Bash environment

## Usage

1. Make sure `APECtyper.sh` is executable:

   ```bash
   chmod +x APECtyper.sh
   ```

2. Run the GUI:

   ```bash
   python apectyper_gui.py
   ```

3. Click **Run APECtyper**

4. Select a FASTA file (`.fa` or `.fasta`)

5. Select an output directory

6. Wait for completion message

---

## Notes

* The GUI is a wrapper and does **not** modify APECtyper itself.
* All outputs are generated by APECtyper in the selected output directory.
* Errors during execution will be shown in a popup window.

---

## Troubleshooting

* **APECtyper.sh not found**
  → Ensure the script is in the same directory as the GUI

* **Permission denied**
  → Run:

  ```bash
  chmod +x APECtyper.sh
  ```

* **Tkinter not found**
  → Install with:

  ```bash
  sudo apt install python3-tk
  ```
